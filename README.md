## 프로젝트 세팅

1. 종속성 설치

```
corepack enable

yarn install
```

2. vscode 확장 설치

   1. vscode 확장 마켓 플레이스 진입
   2. @recommand 입력 후 확장 설치

## 실습

1. branch intro : 다수의 채팅방 없이 구현한 하나의 채팅방
2. branch room : 다수의 채팅방 입장과 각 채팅장 별 채팅

## 실행

2개의 터미널 띄우고 각각 실행

- `yarn server dev`
- `yarn client dev`
